## Goal
* More Easily Code/Read/Maintain E2E Test

* High-Performance

## Features
* Steps controller
* E2E Test Report
* Support CLI
* Compatible with Selenium-webdriver
* Serial perform case
* Auto-create test file with AC or test case
* Auto-modify ticket status from Jira
